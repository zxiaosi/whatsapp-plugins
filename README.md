## 启动说明

  + 安装依赖

  ```bash
    npm install
  ```

  + 启动项目

  ```bash
    npm run dev
  ```

  + 加载插件

  1. 打开 Chrome 浏览器，输入 `chrome://extensions/` 进入插件管理页面 (edge 浏览器输入 `edge://extensions/`)

  2. 打开开发者模式

  3. 点击 `加载已解压的扩展程序`，选择项目中的 `dist` 目录
